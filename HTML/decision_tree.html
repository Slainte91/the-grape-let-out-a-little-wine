Decision Tree and Random Forest

We used a machine learning Decision Tree to figure out which features were most important in determing "good quality" wine. First, we converted the 'Quality' column into a new column where wines rated 7 and over were classified as Good Quality, and given a rating of 1. Wines rated 6 and under were given a 0. Using Jupyter Notebook, we ran a Decision Tree and Random Forest Classifier.


Next, we ran the Decision Tree and Random Forest. The Decision Tree was able to predict Good Quality at 88.5% of the time, and the Random Forest was at 90%.



Finally, we ran a line of code to see which features were most important.
